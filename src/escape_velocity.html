Escape Velocity is a game that ended up defining my adolescence.

Around the time that Escape Velocity: Nova was being developed, Ambrosia Software had web forums and an IRC server. I connected with several very smart, very cool people there. But mostly I remember the spirit of building things to show each other and help each other out.

How to Play

Escape Velocity was originally written for Classic Mac OS, so it's hard to play on modern computers. Luckily, the Endless Sky project is re-implementing the same gameplay in a modern engine.

But that's not the only game engine in town. Another team is building the Kestrel engine for 2D top-down space games.

And there's a Kickstarter called Cosmic Frontier that re-implements the first sequel, Escape Velocity: Override.

If you're into retrocomputing you can still find archives of a bunch of old games from Ambrosia Software.
